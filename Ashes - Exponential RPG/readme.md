## 🎯 Objectifs des guides

### 🔹 **Guide #1**
- **Objectif** : Battre le **premier boss** et préparer les fondations pour les runs suivants.
- **Points clés** :
  - Optimisation des statistiques (Half Power, Half Def, All HP).
  - Acquisition des équipements de base (arme et armure).
  - Préparation pour les runs avancés.

### 🔹 **Guide #2**
- **Objectif** : Battre le **(troisième) boss** et optimiser votre équipement pour des runs plus puissants.
- **Points clés** :
  - Montée en puissance avec des zones de haut niveau.
  - Acquisition d'équipements avancés (Axe, Spear, Gem Def, etc.).
  - Optimisation des statistiques (All Luck) pour maximiser les drops.

## 📖 Comment utiliser ces guides ?
1. **Lisez le [Guide #1](guide-1.md)** pour préparer votre première partie et battre le premier boss.
2. **Suivez le [Guide #2](guide-2.md)** pour progresser vers des zones plus élevées et battre le troisième boss.
